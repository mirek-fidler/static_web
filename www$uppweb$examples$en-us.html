1. Introduction

We would like to present to you a complete collection of examples. All examples are written in the pure U++ framework. This is the best place to raise your skills in using our library. Alternatively, if you still hesitate to use the U++ framework in your own projects, this site should solve your doubts. It will show you on how many real world scenario U++ can be used.

The whole page is divided into two sections. The first one "Sample applications" shows the complex usage of the framework. in the real world application. The second "Reference examples" demonstrates how certain features are supposed to be used.

You can click on each example to display its source code with additional information such as:

a short description that explains what the application is doing

the screen-shot of a working application

All source code packages that are listed on this site are available in standard U++ distribution. So, you can easily select example that you are interested in and build it in TheIDE.
